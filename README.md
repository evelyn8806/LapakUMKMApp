SDGs yang dipilih : 
- SDG 1 : No Poverty (Tanpa Kemiskinan)
Aplikasi LapakUMKM membantu pelaku UMKM memperoleh akses yang lebih mudah untuk mendapatkan lapak usaha sehingga dapat meningkatkan kesempatan memperoleh penghasilan.
- SDG 8 : Decent Work and Economic Growth (Pekerjaan Layak dan Pertumbuhan Ekonomi)
Aplikasi ini mendukung pertumbuhan UMKM dengan mempermudah proses pencarian, pengajuan, dan pengelolaan penyewaan lapak secara digital sehingga dapat menciptakan peluang usaha dan meningkatkan kesejahteraan masyarakat.

Deskripsi singkat:
LapakUMKM adalah aplikasi Android yang dirancang untuk membantu pelaku UMKM mencari, mengajukan penyewaan, dan mengelola lapak usaha secara efisien. Selain itu, aplikasi menyediakan fitur bagi admin untuk mengelola data pengguna, pengajuan penyewaan, serta proses pembayaran.

Fitur utama :
Pengguna
- Login dan Registrasi
- Dashboard
- Melihat daftar lapak
- Pengajuan penyewaan lapak
- Pembayaran
- Profil pengguna
  
Admin
- Dashboard Admin
- Mengelola data pengguna
- Melihat dan memproses pengajuan penyewaan
- Melihat detail pengajuan
- Monitoring aktivitas penyewaan

Cara menjalankan :
- User
1. Daftar akun pelaku UMKM dan login
2. Pada halaman dashboard, pilih lapak yang diinginkan
3. Pengisian data untuk melakukan pengajuan penyewaan
4. pembayaran dan menunggu persetujuan
5. jika telah disetujui dapat melihat QR / E-Tiket
- Admin
1. daftar akun admin (email harus @umkm.com) dan login
2. pada halaman dashboard dapat melihat pengajuan baru
3. review data pelaku UMKM
4. menyetujui atau menolak pengajuan
